#  Розробка комп’ютерної гри жанру соціальної дедукції “Мафія”
Кваліфікаційна робота на підтвердження ступеня фахового молодшого
бакалавра ([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)). Керівник
роботи – **Кривцова Олена Павлівна**.
## Завдання до дипломної роботи
Розроблений програмний продукт має надавати всю необхідну функціональність, яка включає автоматичну генерацію номерів гравців, випадковий розподіл ролей, оголошення хвилини мафії, відлік часу для промови кожного гравця та повідомлення про перехід гри у нічну фазу. Особливу увагу слід звернути на аналіз аналогічних продуктів, дослідження предметної області завдання та вимог користувачів. Мова реалізації – С#. Розроблений додаток та його структура мають бути детально описані у пояснювальній записці.
## Використані технології
* C# — основна мова програмування для реалізації логіки програмного продукту;
* Unity — середовище розробки та ігровий рушій для створення застосунку;
* TextMeshPro (TMP) — бібліотека для відображення текстової інформації в інтерфейсі користувача;
* Unity UI — засоби створення графічного інтерфейсу (кнопки, текстові поля та інші елементи керування);
* AudioSource — компонент Unity для відтворення звукових повідомлень і сигналів;
* Coroutines (IEnumerator) — механізм Unity для реалізації таймерів, затримок та послідовного виконання сценаріїв гри;
* draw.io (diagrams.net) — інструмент для створення UML-діаграм, діаграми прецедентів, діаграми класів, діаграми діяльності та діаграми послідовності;
* Microsoft Visual Studio — середовище розробки для написання, налагодження та тестування програмного коду;
* GitHub (якщо використовувала) — система контролю версій для зберігання та керування вихідним кодом проєкту.
